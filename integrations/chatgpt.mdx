# ChatGPT + Nitrosend MCP

ChatGPT has native MCP support, giving you the full Nitrosend toolkit — 19 tools, live brand context, and account state — directly in your chat.

## Setup

### Step 1: Add the MCP server

In ChatGPT, go to **Settings → Connectors → Add MCP server** and enter:

```text
https://api.nitrosend.com/mcp
```

### Step 2: Sign in

ChatGPT will open a Nitrosend sign-in window. Log in or create a free account — no API key needed.

### Step 3: Use it

ChatGPT now has access to all 19 Nitrosend MCP tools. Ask it things like:

- *"Check my Nitrosend account status"*
- *"Create an email campaign about our summer sale"*
- *"Show me the performance of my last campaign"*
- *"Draft a welcome flow for new subscribers"*

<Tip>
  MCP gives ChatGPT your full brand context — colors, fonts, tone of voice — so emails it creates are on-brand without extra prompting.
</Tip>

## Example workflows

### Compose a campaign

```
Create a campaign called "Product Launch" targeting my subscriber list.
Include a hero section with our logo, feature highlights in columns,
and a CTA button linking to our landing page.
```

### Build a flow

```
Create a welcome flow: when a new contact is added, send a welcome
email immediately, wait 3 days, then send a follow-up with our
top recommendations.
```

### Check analytics

```
Show me open and click rates for my last 5 campaigns. How do they
compare to benchmarks?
```

## Custom GPT with Actions (Alternative)

For building a shareable GPT that uses the Nitrosend REST API directly:

### 1. Create a custom GPT

Go to [ChatGPT](https://chat.openai.com) → **Explore GPTs** → **Create a GPT**.

### 2. Add the Nitrosend action

In the GPT editor, go to **Configure** → **Actions** → **Create new action**.

Set the **Authentication** to:
- Type: **API Key**
- Auth Type: **Bearer**
- Key: your `nskey_live_...` key (from Settings > API Keys in the [Nitrosend dashboard](https://app.nitrosend.com))

Import the OpenAPI schema from:

```
https://api.nitrosend.com/openapi.yaml
```

### 3. Add instructions

In the GPT's **Instructions** field:

```
You are an email marketing assistant connected to Nitrosend.
You can create campaigns, manage contacts, build automation flows,
and check analytics. Always confirm before sending campaigns.
Ask for a preview or test send before going live.
```

## Key API endpoints

| Action | Method | Endpoint |
|--------|--------|----------|
| Send a message | `POST` | `/v1/my/messages` |
| List campaigns | `GET` | `/v1/my/campaigns` |
| Create a campaign | `POST` | `/v1/my/campaigns` |
| List contacts | `GET` | `/v1/my/contacts` |
| Import contacts | `POST` | `/v1/my/contacts/import` |
| List templates | `GET` | `/v1/my/templates` |
| Get account info | `GET` | `/v1/my/account` |

See the full [API Reference](/api-reference) for all available endpoints.

<Info>
  MCP with OAuth is the recommended setup — no API keys to manage. The Custom GPT method is an alternative for building shareable GPTs.
</Info>
