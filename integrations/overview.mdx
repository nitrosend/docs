# Connect Your AI Tool

Nitrosend is designed to be controlled by AI. Instead of clicking through dashboards, you connect your AI tool and manage everything — campaigns, flows, contacts, templates, analytics — through natural language.

## MCP: The Fastest Way to Connect

**MCP (Model Context Protocol)** is an open standard that lets AI assistants connect directly to external tools. Nitrosend has a native MCP server, which means Claude, Cowork, and other MCP-compatible clients can access your full email marketing platform with zero coding.

### What MCP gives your AI

When you connect via MCP, your AI assistant gets:

- **20 specialized tools** — compose emails, build flows, manage contacts, check analytics, control delivery, and more
- **Live resources** — your brand identity, account state, platform guide, and email design schema
- **Full context** — your AI knows your brand colors, tone of voice, and sending history before it writes a single email
- **Two-way control** — create, preview, test, approve, send, pause, and analyze, all from conversation

### MCP vs REST API

| | MCP | REST API |
|---|---|---|
| **Best for** | Claude, ChatGPT, Cursor, Codex | Custom apps, Gemini, Zapier, n8n |
| **Setup** | 1 click or 1 command, OAuth sign-in | API key + define function calls or curl |
| **AI context** | Full — brand, schema, guide, account state | You provide context manually |
| **Tools** | 20 purpose-built tools with validation | Raw HTTP endpoints |
| **Coding required** | No | Minimal to moderate |

<Tip>
  **Use MCP whenever possible.** It gives your AI far more context about your account, brand, and email best practices than the REST API alone. The result is better emails with less prompting.
</Tip>
