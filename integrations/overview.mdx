# Connect Your AI Tool

Nitrosend is designed to be controlled by AI. Instead of clicking through dashboards, you connect your AI tool and manage everything — campaigns, flows, contacts, templates, analytics — through natural language.

## MCP: The Fastest Way to Connect

**MCP (Model Context Protocol)** is an open standard that lets AI assistants connect directly to external tools. Nitrosend has a native MCP server, which means Claude, ChatGPT, Codex, Gemini, and other MCP-compatible clients can access your full email marketing platform with zero coding.

### What MCP gives your AI

When you connect via MCP, your AI assistant gets:

- **19 specialized tools** — compose emails, build flows, manage contacts, check analytics, control delivery, and more
- **Live resources** — your brand identity, account state, platform guide, and email design schema
- **Full context** — your AI knows your brand colors, tone of voice, and sending history before it writes a single email
- **Two-way control** — create, preview, test, approve, send, pause, and analyze, all from conversation

### MCP vs REST API

| | MCP | REST API |
|---|---|---|
| **Best for** | Claude, ChatGPT, Codex, Gemini, and any MCP client | Custom apps, webhooks, automation platforms |
| **Setup** | 1 click or 1 config entry | Define function calls or use curl |
| **AI context** | Full — brand, schema, guide, account state | You provide context manually |
| **Tools** | 19 purpose-built tools with validation | Raw HTTP endpoints |
| **Coding required** | No | Minimal to moderate |

<Tip>
  **Use MCP whenever possible.** It gives your AI far more context about your account, brand, and email best practices than the REST API alone. The result is better emails with less prompting.
</Tip>

## Authentication

**MCP connections use OAuth** — your AI tool opens a browser for sign-in. No API key needed. This is the recommended setup for all platforms.

**REST API** uses API keys (`nskey_live_...`) for programmatic access. Generate one from **Settings > API Keys** in the [Nitrosend dashboard](https://app.nitrosend.com).

<Warning>
  Keep API keys secret. Never expose them in client-side code, commit them to version control, or share them publicly. For interactive AI tools, use MCP with OAuth instead.
</Warning>
