# AI & API Integrations

Nitrosend is built to work with AI. Whether you're using a chat assistant, a coding tool, or building your own integration, Nitrosend provides two ways to connect:

<CardGroup cols={2}>
  <Card title="MCP Server" icon="plug">
    A native Model Context Protocol server that lets AI assistants manage your email marketing directly — no code required.
  </Card>
  <Card title="REST API" icon="square-terminal">
    A standard REST API with Bearer token auth for any language, platform, or automation tool.
  </Card>
</CardGroup>

## Which method should I use?

| Method | Best for | Setup time |
|--------|----------|------------|
| **MCP Server** | Claude, Claude Code, Claude Cowork, and any MCP-compatible client | 1 minute |
| **REST API** | GPT, Gemini, Codex, custom apps, Zapier, n8n, or any HTTP client | 5 minutes |

## Get your API key

Both methods use the same API key. Generate one from **Settings > API Keys** in the [Nitrosend dashboard](https://app.nitrosend.com).

API keys are prefixed with `nskey_live_` for production and `nskey_test_` for test mode.

<Warning>
  Keep your API key secret. Never expose it in client-side code, commit it to version control, or share it publicly.
</Warning>
