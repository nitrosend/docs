## Quick facts

- **Base URL**: `https://api.nitrosend.com/v1/my`
- **Auth**: `Authorization: Bearer nskey_live_...` — grab a key in **Settings > API Keys** at [app.nitrosend.com](https://app.nitrosend.com)
- **OpenAPI spec**: [`https://api.nitrosend.com/openapi.yaml`](https://api.nitrosend.com/openapi.yaml) — import it into Postman, an SDK generator, or your agent's tool schema

<Info>
  Every plan, including free plans, has full API access. Plan limits apply to send volume, seats, and AI action allowances — not to the API itself.
</Info>
