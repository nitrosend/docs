# Google Gemini

You can connect Nitrosend to Google Gemini using function calling. This lets Gemini manage your email marketing by calling the Nitrosend REST API on your behalf.

## How it works

Gemini's function calling feature lets you define external tools that the model can invoke. You define Nitrosend operations as function declarations, and when Gemini decides to use one, your code executes the corresponding API call and returns the result.

## Setup with Google AI Studio

### 1. Get your API key

Go to **Settings > API Keys** in the [Nitrosend dashboard](https://app.nitrosend.com) and copy your key.

### 2. Define function declarations

```python
from google import genai
from google.genai import types

client = genai.Client()

# Define Nitrosend tools as function declarations
nitrosend_tools = types.Tool(
    function_declarations=[
        types.FunctionDeclaration(
            name="send_email",
            description="Send a transactional email to one recipient immediately",
            parameters=types.Schema(
                type=types.Type.OBJECT,
                properties={
                    "to": types.Schema(type=types.Type.STRING, description="Recipient email"),
                    "subject": types.Schema(type=types.Type.STRING, description="Subject line"),
                    "body": types.Schema(type=types.Type.STRING, description="Email body (HTML)")
                },
                required=["to", "subject", "body"]
            )
        ),
        types.FunctionDeclaration(
            name="list_campaigns",
            description="List all email campaigns with their status",
            parameters=types.Schema(
                type=types.Type.OBJECT,
                properties={
                    "status": types.Schema(
                        type=types.Type.STRING,
                        description="Filter by status: draft, active, paused, completed"
                    )
                }
            )
        ),
        types.FunctionDeclaration(
            name="create_contact",
            description="Add a new contact to the audience",
            parameters=types.Schema(
                type=types.Type.OBJECT,
                properties={
                    "email": types.Schema(type=types.Type.STRING, description="Contact email"),
                    "first_name": types.Schema(type=types.Type.STRING, description="First name"),
                    "last_name": types.Schema(type=types.Type.STRING, description="Last name"),
                    "opt_in": types.Schema(type=types.Type.BOOLEAN, description="Subscribe for email delivery")
                },
                required=["email"]
            )
        ),
        types.FunctionDeclaration(
            name="get_account_status",
            description="Get account health, onboarding status, and sending limits",
            parameters=types.Schema(type=types.Type.OBJECT, properties={})
        )
    ]
)
```

### 3. Handle function calls

```python
import requests

NITROSEND_API_KEY = "nskey_live_your_key_here"
BASE_URL = "https://api.nitrosend.com/v1/my"
HEADERS = {
    "Authorization": f"Bearer {NITROSEND_API_KEY}",
    "Content-Type": "application/json"
}

def execute_nitrosend_function(name, args):
    """Route Gemini function calls to Nitrosend API endpoints."""

    if name == "send_email":
        resp = requests.post(f"{BASE_URL}/messages", headers=HEADERS, json={
            "channel": "email",
            "to": args["to"],
            "subject": args["subject"],
            "body": args["body"]
        })
        return resp.json()

    elif name == "list_campaigns":
        params = {"status": args["status"]} if "status" in args else {}
        resp = requests.get(f"{BASE_URL}/campaigns", headers=HEADERS, params=params)
        return resp.json()

    elif name == "create_contact":
        resp = requests.post(f"{BASE_URL}/contacts", headers=HEADERS, json=args)
        return resp.json()

    elif name == "get_account_status":
        resp = requests.get(f"{BASE_URL}/account", headers=HEADERS)
        return resp.json()
```

### 4. Run the conversation

```python
import json

response = client.models.generate_content(
    model="gemini-2.0-flash",
    contents="How many campaigns do I have?",
    config=types.GenerateContentConfig(tools=[nitrosend_tools])
)

# Check if Gemini wants to call a function
for part in response.candidates[0].content.parts:
    if part.function_call:
        call = part.function_call
        result = execute_nitrosend_function(call.name, dict(call.args))

        # Send the result back to Gemini
        followup = client.models.generate_content(
            model="gemini-2.0-flash",
            contents=[
                types.Content(parts=[part]),
                types.Content(parts=[
                    types.Part(function_response=types.FunctionResponse(
                        name=call.name,
                        response={"result": json.dumps(result)}
                    ))
                ])
            ],
            config=types.GenerateContentConfig(tools=[nitrosend_tools])
        )
        print(followup.text)
```

## Using with Google AI Studio (no code)

You can also test Nitrosend integration in [Google AI Studio](https://aistudio.google.com):

1. Open AI Studio and start a new prompt
2. Click **Tools** → **Function Calling**
3. Add function declarations matching the Nitrosend API endpoints you want to use
4. The model will generate function calls you can execute manually or wire up to your backend

## Key API endpoints for Gemini

| Function name | API call | What it does |
|---------------|----------|-------------|
| `send_email` | `POST /v1/my/messages` | Send a single email immediately |
| `list_campaigns` | `GET /v1/my/campaigns` | List all campaigns |
| `create_campaign` | `POST /v1/my/campaigns` | Create a new campaign |
| `list_contacts` | `GET /v1/my/contacts` | List contacts in your audience |
| `create_contact` | `POST /v1/my/contacts` | Add a new contact |
| `get_account` | `GET /v1/my/account` | Check account status and limits |

See the full [API Reference](/api-reference) for all available endpoints.

<Info>
  Gemini's function calling works with both `gemini-2.0-flash` and `gemini-2.5-pro`. Use flash for speed, pro for complex multi-step marketing tasks.
</Info>
