# OpenAI Codex CLI + Nitrosend MCP

[Codex CLI](https://github.com/openai/codex) is OpenAI's terminal-based coding agent. Codex has native MCP support with OAuth, so you get the full Nitrosend toolkit — 19 tools, live brand context, and account state — with a single command.

## Setup

### Option A: OAuth (Recommended)

No API key needed. Codex handles sign-in via your browser.

**Via CLI:**

```bash
codex mcp add nitrosend --url https://api.nitrosend.com/mcp
codex mcp login nitrosend
```

**Or add to `~/.codex/config.toml`:**

```toml
[mcp_servers.nitrosend]
url = "https://api.nitrosend.com/mcp"
```

Then run `codex mcp login nitrosend` to authenticate. Codex opens your browser — sign in or create a free Nitrosend account.

### Option B: API Key

For non-interactive or CI environments, use an API key instead.

Add your API key to your shell profile (`~/.zshrc`, `~/.bashrc`, etc.):

```bash
export NITROSEND_API_KEY="nskey_live_your_key_here"
```

Then add to `~/.codex/config.toml`:

```toml
[mcp_servers.nitrosend]
url = "https://api.nitrosend.com/mcp"
bearer_token_env_var = "NITROSEND_API_KEY"
```

Get your API key from **Settings > API Keys** in the [Nitrosend dashboard](https://app.nitrosend.com).

### Verify

Start Codex and try:

```bash
codex "Check my Nitrosend account status"
```

Codex calls the `nitro_get_status` tool and shows your account health, onboarding progress, and recommendations.

<Tip>

## Project-scoped config

You can also add Nitrosend to a project-scoped `.codex/config.toml` in trusted projects. This keeps the config alongside your codebase.

## Troubleshooting

<AccordionGroup>
  <Accordion title="OAuth login doesn't open browser">
    Run `codex mcp login nitrosend` manually. If your environment doesn't support browser launch, use the API key method instead.
  </Accordion>
  <Accordion title="Tool calls time out">
    Brand scraping can take 20-30 seconds. If other calls time out, check your network connection to `api.nitrosend.com`.
  </Accordion>
</AccordionGroup>

<Info>
  OAuth is recommended — no secrets to manage, tokens refresh automatically.
</Info>
