# Templates

Templates are reusable email designs. Build them once with the drag-and-drop editor, then use them across campaigns and flows.

## Editor

The visual editor uses a section-based layout. Drag components (text, images, buttons, dividers) into sections. Your brand theme is applied automatically.

## Personalization

Use merge tags to personalize content:

```
Hi {{ contact.first_name }},
```

## MJML rendering

Templates are stored as structured JSON and rendered to MJML for maximum email client compatibility. You don't need to write any HTML.
