# Building Flows

Flows are automated sequences of actions triggered by events. Use them for welcome series, abandoned cart reminders, re-engagement campaigns, and more.

## How flows work

A flow starts with a **trigger** — an event that kicks off the automation. Then it executes a series of **steps**: sending emails, waiting for a delay, splitting by condition, and more.

## Creating a flow

1. Go to **Flows > New Flow**
2. Choose a trigger (e.g., "Contact added to list", "Event received")
3. Add steps using the visual flow editor
4. Activate the flow

## Step types

- **Send Email** — send a template to the contact
- **Send SMS** — send a text message
- **Wait** — pause for a duration (e.g., 2 days)
- **Condition** — branch based on contact properties or behavior
- **Update Contact** — modify contact fields

## Testing

Use the **Test** button to simulate a flow run with a test contact before activating.
